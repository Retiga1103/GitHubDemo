# ClothesWasher

A simple Java program that simulates the operation of a washing machine. The `ClothesWasher` class includes attributes and methods to represent the washing machine's functionality, such as turning it on, setting the washing mode, starting the washing process, and turning it off.

## Features

- **Attributes**:
  - `brand`: The brand of the washing machine.
  - `capacity`: The capacity of the washing machine in kilograms.
  - `mode`: The washing mode (e.g., "Normal", "Delicate", "Heavy").
  - `isOn`: A boolean to track whether the washing machine is on or off.

- **Methods**:
  - `turnOn()`: Turns the washing machine on.
  - `setMode(String mode)`: Sets the washing mode.
  - `startWashing()`: Simulates the washing process.
  - `turnOff()`: Turns the washing machine off.
